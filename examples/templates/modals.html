Modals

The best way to invoke the modal window is through directive sd-modal.

Modal state (open/close) is handled by data-model="" attribute.

Default

By default modal has 480px width and auto height (depends on the amount of content).

Modal with min height modifier

Sometimes there is a need to add min height to modal body, and in that case add class .modal__body--add-min-height on div block .modal__body. This will add min height of 300px to modal body.

Fullscreen

Fullscreen modal behaves as full page overlay, width and height set to 100%. Add class .modal--fullscreen on div sd-modal. There is also class .modal--white that applies white background to modal body.

Tabs

Special case for modals with tabbed navigation and nested footer inside the body. Add class .modal--tabs on div sd-modal. This version also works with different modal size, for example just add .modal--large to get large modal with tabs.

Modal with external template

In some cases we will need to trigger modal through angular with defined template in different .html file. This can be done with sdModalService service.

Example Markup 
                 var modal;
                scope.openModal = function () {
                    modal = sdModalService.open({
                        scope: scope,
                        controller: ctrlModalController,
                        controllerAs: 'ctrl',
                        size: 'fullscreen',
                        classes: 'modal--white',
                        template: 'templates/modal-template.html'
                    });
                };
                scope.close = function () {
                    modal.close();
                };
